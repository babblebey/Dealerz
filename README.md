## Install
Run the Command to Install all dependencies.
```sh
yarn 
```
or
```sh
yarn install
```

## Usage

Now run the command to start the development server.
```sh
yarn dev
```
or
```sh
yarn run dev
```

Your project should start running on `http://localhost:5173` or `http://127.0.0.1:5173/`
